## Whenever downloading dependencies

### `npm install --save`
saves the new dependencies to the package manager. 

# How to Start the program

### `npm run dev`

# When Pushing Code:

### `npm run lint`
Run linter to see build errors

### `npm run fix`
Fix linter errors

Build needs to pass linter errors in order to merge with master.
